# 🚀 Groq with LangChain

Welcome to the **Groq with LangChain** repository! This project demonstrates how to integrate high-speed inference models using **Groq** alongside **LangChain** for building powerful AI applications in Python.

---

## 📌 Features

* ⚡ **High-Speed Inference**: Connect seamlessly to Groq's fast LLM infrastructure.
* 🦜🔗 **LangChain Integration**: Leverage LangChain abstractions (`ChatGroq`) for structured prompts and chains.
* 🔐 **Secure Configuration**: Uses `python-dotenv` to keep API keys safely managed outside the source code.
